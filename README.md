# java-jdbc 的实现

## 一、介绍

本项目实现了 Java 的 JDBC 。

**初衷**：学习-体验-成长。

​		自己学习 Java 过程中对于知识点的强化记忆，强化代码练习；方便回顾和实用。

**项目内容：**项目从***原生***到***封装***最后到***封装成工具类***依次实现
